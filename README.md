# E-Commerce Product Management API

## 🚀 Tech Stack
- Java
- Spring Boot
- Spring Data JPA (Hibernate)
- MySQL
- REST API

## 📌 Features
- CRUD operations for products
- Image upload and retrieval
- Keyword-based product search (JPQL)
- Layered architecture (Controller, Service, Repository)

## 📷 API Endpoints
- GET /api/products
- GET /api/product/{id}
- POST /api/product
- PUT /api/product/{id}
- DELETE /api/product/{id}

## 💡 Description
This project is a backend REST API built using Spring Boot to manage e-commerce products efficiently with database integration and image handling.
